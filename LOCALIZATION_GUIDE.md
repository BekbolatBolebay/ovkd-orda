# Қазақ - Орыс тіліне локализация нұсқалығы / Руководство по локализации на казахский-русский языки

## Обзор / Қарама-қарсы түсіндіру

Сайт поддерживает двуязычное переключение между казахским (kz) и русским (ru) языками. Языковой переключатель (KZ | RU) расположен в заголовке сайта.

## Архитектура локализации / Локализация архитектурасы

### 1. Основные компоненты / Негізгі компоненттер

- **`lib/i18n.ts`** - Центральный файл переводов всех строк заголовка и стандартных фраз
- **`components/language-provider.tsx`** - React Context Provider для управления языком (использует localStorage)
- **`components/language-switcher.tsx`** - UI компонент для переключения языков (KZ | RU)
- **`components/translated-text.tsx`** - Вспомогательные компоненты для перевода текста

### 2. Как использовать локализацию в компонентах / Компонентілерде локализацияны пайдалану

#### Вариант 1: Использование `useLanguage()` хука

```tsx
"use client"

import { useLanguage } from "@/components/language-provider"

export function MyComponent() {
  const { language } = useLanguage()

  return (
    <div>
      <h1>
        {language === 'ru'
          ? 'Добро пожаловать на наш сайт'
          : 'Біздің сайтына қош келдіңіз'}
      </h1>
    </div>
  )
}
```

#### Вариант 2: Использование компонента `TranslatedText`

```tsx
"use client"

import { TranslatedText } from "@/components/translated-text"

export function MyComponent() {
  return (
    <div>
      <h1>
        <TranslatedText
          kz="Біздің сайтына қош келдіңіз"
          ru="Добро пожаловать на наш сайт"
        />
      </h1>
    </div>
  )
}
```

#### Вариант 3: Использование хука `useTranslation()` для глобальных строк

```tsx
"use client"

import { useTranslation } from "@/components/language-provider"

export function Header() {
  const t = useTranslation()

  return (
    <nav>
      <a href="/news">{t?.header?.nav?.newsBtn}</a>
      <a href="/contact">{t?.header?.nav?.contactBtn}</a>
    </nav>
  )
}
```

## Добавление новых переводов / Жаңа аудармаларды қосу

### Шаг 1: Добавить в `lib/i18n.ts`

```typescript
export const TRANSLATIONS = {
  kz: {
    // Существующие переводы...
    pages: {
      myPage: {
        title: "Менің беті",
        description: "Бұл менің бетімнің сипаттамасы"
      }
    }
  },
  ru: {
    // Существующие переводы...
    pages: {
      myPage: {
        title: "Моя страница",
        description: "Это описание моей страницы"
      }
    }
  }
}
```

### Шаг 2: Использовать в компоненте

```tsx
const t = useTranslation()
// или
const { language } = useLanguage()

return (
  <h1>{language === 'ru' ? t?.pages?.myPage?.title_ru : t?.pages?.myPage?.title}</h1>
)
```

## Примеры переведённых страниц / Аударылған беттердің мысалдары

### Home Page (Үй беті)
✅ Полностью переведена с использованием `useLanguage()` хука
- Новости (СОҢҒЫ ЖАҢАЛЫҚТАР / ПОСЛЕДНИЕ НОВОСТИ)
- Кнопки действий (ФОТОБАЯН/ФОТОГАЛЕРЕЯ и т.д.)
- Статистика (Жыл тәжірибе / Лет опыта)
- Документы

### Header (Хедер)
✅ Полностью переведён
- Верхние ссылки (Науқастарға / Пациентам)
- Навигация (Хедер), включая все меню
- Язык в документе автоматически обновляется

## Рекомендации по локализации других страниц / Басқа беттерді локализациялау бөлігіндегі сәтті ынамдау

1. **Сделайте компонент `"use client"`**, если нужен доступ к `useLanguage()`
2. **Используйте паттерн условного рендеринга**:
   ```tsx
   <h1>{language === 'ru' ? 'Русский текст' : 'Қазақ мәтіні'}</h1>
   ```
3. **Извлекайте часто используемые строки в `lib/i18n.ts`**
4. **Тестируйте оба языка** после внесения изменений

## Сохранение языка / Тілді сақтау

Выбранный язык автоматически сохраняется в `localStorage` под ключом `website-language`. Язык восстанавливается при следующем посещении сайта.

## Отладка / Іздеу

Если локализация не работает:

1. Проверьте, что компонент имеет `"use client"` директиву
2. Убедитесь, что компонент обёрнут в `LanguageProvider`
3. Проверьте консоль браузера на ошибки
4. Очистите локальное хранилище: `localStorage.clear()`
5. Перезагрузите страницу

## Быстрая начало / Тез тәжіринің бөлініңіз

Для добавления простого перевода на страницу:

```tsx
"use client"

import { useLanguage } from "@/components/language-provider"

export default function MyPage() {
  const { language } = useLanguage()

  return (
    <main>
      <h1>
        {language === 'ru'
          ? 'Заголовок на русском'
          : 'Қазақша тақырыбы'}
      </h1>
      <p>
        {language === 'ru'
          ? 'Описание на русском'
          : 'Қазақша сипаттамасы'}
      </p>
    </main>
  )
}
```
